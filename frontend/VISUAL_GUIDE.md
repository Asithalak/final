## Quick Reference: CSS Classes

### Hover Effects
- `group` - Parent container
- `group-hover:opacity-100` - Show on parent hover
- `group-hover:scale-105` - Zoom effect
- `group-hover:translate-y-0` - Slide up effect

### Transitions
- `transition-opacity duration-300` - Fade transitions
- `transition-transform duration-300` - Movement transitions
- `transition-all duration-200` - General transitions

### Animations
- `animate-pulse` - Arrow pulse effect
- `animate-fadeIn` - Custom fade in
- `animate-slideUp` - Custom slide up
